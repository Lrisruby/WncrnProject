# Toycar.My childhood

  应用介绍:"Toycar.My childhood"app是一款线上购买儿童车/玩具车的应用。

  应用功能:
  
    1、首页: 儿童电动车/扭扭车/儿童自行车/滑板车、店内热销、推荐专区等类型；
    
    2、分类: 滑板车、儿童自行车、儿童电动车、扭扭车等类型商品；
    
    3、兑换专区: 积分明细/兑换记录、以及兑换列表等；
    
    3、购物车: 全场商品均可加入购物车，购物车可直接选择商品进行结算；
    
    4、我的: 优惠券、我的积分、我的订单、我的地址和我的收藏等。
